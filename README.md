# Panduan Evaluasi Kinerja Pegawai ASN — Triwulan II 2026

Website panduan internal berbasis satu file HTML, dibuat mengikuti gaya struktur
`panduan-mbcl.netlify.app` (sidebar navigasi per bab, konten interaktif single-page),
diisi dengan materi dari Surat Sekretaris Jenderal Kemenkes Nomor KP.02.04/A/14795/2026
tanggal 2 Juli 2026 perihal Evaluasi Kinerja Pegawai ASN Kemenkes Periode Triwulan II
Tahun 2026 beserta 3 lampirannya.

## Isi Folder

- `index.html` — website panduan lengkap, siap dibuka langsung di browser (double-click)
  atau di-deploy ke Vercel/Netlify/Replit tanpa build step apapun.
- `konten.md` — sumber konten dalam format Markdown (14 bab), untuk referensi/edit cepat
  tanpa perlu buka HTML.
- `README.md` — file ini.

## Cara Pakai

### Lihat langsung
Buka `index.html` di browser mana saja — tidak butuh server atau instalasi.

### Deploy ke Netlify/Vercel (opsional)
1. Push folder ini ke repo GitHub baru (mis. `panduan-evaluasi-skp`).
2. Import repo di Netlify atau Vercel — tidak perlu build command, cukup arahkan ke
   folder ini sebagai root (`index.html` di root).
3. Deploy → dapat URL publik untuk dibagikan ke tim.

## Struktur Bab

1. **Ringkasan** — ikhtisar surat & angka-angka kunci
2. **Ketentuan Kinerja** — 5 predikat, anomali TW1, komposisi distribusi
3. **Proses Evaluasi** — tahapan evaluasi, konfirmasi & keberatan, dialog kinerja & tunjangan
4. **Panel Kalibrasi** — ketentuan umum, peserta & tahapan, fokus pembahasan, tabel korelasi K1/K2/K3
5. **Referensi** — contoh surat keterangan, daftar pejabat yang dituju, garis waktu
